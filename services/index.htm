Rich internet applications

We specialize in developing rich internet applications (RIAs); "web applications that have the features and functionality of traditional desktop applications" [Wikipedia]. Our core technologies for rich client development are Adobe Flex and Flash. We also have experience with Ajax technologies and the Microsoft Silverlight platform.

RIAs, with a service-oriented architecture, are a core aspect of Web 2.0.

Websites

We consider standards compliance an important part of web site development and design and build sites that are WAI compliant using CSS. We also build sites that are optimized for search engine crawlers.

Content Management Systems

We are experienced in implementing web site content management systems (CMS’s) ranging from Adobe Contribute, an entry-level static HTML page editor, to more sophisticated open source products such as Umbraco and DotNetNuke.

Web applications

We have built web applications using a variety of different technologies, including PHP and Classic ASP. Our core technology for developing web applications is the Microsoft .NET Framework.

Online marketing

We have developed a broad range of animated banner placements and games for online marketing purposes. We are also experienced in coordinating campaigns through the Eyeblaster advertising platform. Our core technology for online marketing is Flash.
